Kodi Addons Repository for Kodi/XBMC Matrix by tveecf.




    

  • Link:  https://tveecf.github.io/
    • 





Add Repository to Kodi


    

  • Navigate to System -> File Manager
    • 

  • Open file browser and click on Add Source
    • 

  • For the path, enter: https://tveecf.github.io/
    • 

  • For the name, enter: tveecf
    





    

  • Next, go to System -> Add-ons
    • 

  • Click on Install from zip file
    • 

  • Navigate to tveecf/repository.bitorbit-1.0.0.zip
    • 

  • Click OK
    • 





    

  • Now back to System -> Add-ons
    • 

  • Click on Install from repository
    • 

  • Select bitorbit repository and browse the program addons for BitMatrix 
    • 





BitMatrix


    

  • open the wizard and navigate to Builds -> Mimik / Head -> Install
    • 

  • then follow the installation process. It could take a minute.
    • 

  • Do not press any buttons during installation.
    •
